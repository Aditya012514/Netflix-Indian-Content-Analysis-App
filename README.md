# 🎬 Netflix Indian Content Analysis App

An interactive Netflix Indian Content dashboard built using Python, Streamlit, Pandas, and Plotly.

This project focuses on exploring and visualizing Indian movies and web series available on Netflix through interactive charts, filters, and recommendations.

---

## 🚀 Features

- Interactive Dashboard
- Search Movies & Web Series
- Filter Content by Type and Release Year
- Genre Analysis
- Content Release Trends
- Recommended Indian Shows & Movies Section
- Custom Dark Netflix-style UI

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

---

## 📂 Dataset

The dataset contains Netflix titles data including:
- Movies
- TV Shows
- Genres
- Release Years
- Ratings
- Country Information

---

## ⚠️ Note

This is a beginner-level data analytics and dashboard project created for learning and portfolio purposes.

It is not a complete production-level streaming application and does not include:
- User Authentication
- Backend Database
- Machine Learning Recommendation Engine
- API Integration
- Real-time Streaming Features
